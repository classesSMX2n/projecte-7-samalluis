# P02: Desplegament de la Web Corporativa 🚀

## 📝 Descripció del Projecte
Aquest apartat correspon a la fase final de presentació per a **FoodLogístics**. Després d'haver seleccionat la proposta tecnològica més adequada a la Tasca 8, procedim al desplegament oficial de la solució per fer-la accessible al client i als responsables de l'empresa.

---

## 🛠️ Passos per al Desplegament

Per tal de garantir que el web sigui visible correctament a través de **GitHub Pages**, cal seguir aquests passos de configuració al repositori:

### 1. Preparació de l'entorn local
* Crea una carpeta anomenada `docs` a l'arrel del teu repositori local.
* Copia tots els arxius de la versió definitiva del web (HTML, CSS, JS, imatges, etc.) dins d'aquesta carpeta.

### 2. Sincronització amb GitHub
Cal pujar els canvis al núvol mitjançant els comandaments de Git:
```bash
git add docs/
git commit -m "Add: Versió definitiva de la web corporativa per a FoodLogístics"
git push origin main
